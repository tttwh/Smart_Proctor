1. 项目简介 (PROJECT DESCRIPTION)
-------------------------------------------------------------------------
本项目实现了一套针对线下机房考试场景的自动化监考系统。利用轻量级的 YOLOv8n 模型，
系统能够实时检测考试中的违规行为（如使用手机、查阅书籍等）。
